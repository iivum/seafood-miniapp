# Category Icons

This directory should contain the following category icon images (120x120px PNG recommended):

| File | Description |
|------|-------------|
| `fish.png` | Fish category icon |
| `shrimp.png` | Shrimp & crab category icon |
| `shell.png` | Shellfish category icon |
| `live.png` | Live seafood category icon |
| `frozen.png` | Frozen seafood category icon |
| `dried.png` | Dried seafood category icon |

Add your icon design files here before running the miniprogram.
