# TabBar Icons

This directory should contain the following icon images (81x81px PNG recommended):

| File | Description |
|------|-------------|
| `home.png` | Home tab (inactive) |
| `home-active.png` | Home tab (active) |
| `category.png` | Category tab (inactive) |
| `category-active.png` | Category tab (active) |
| `cart.png` | Cart tab (inactive) |
| `cart-active.png` | Cart tab (active) |
| `profile.png` | Profile tab (inactive) |
| `profile-active.png` | Profile tab (active) |

Add your icon design files here before running the miniprogram.
